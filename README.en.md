# Chalkline Board

Chalkline Board is a local-first classroom whiteboard for teaching, annotation, blank writing, and image explanation. It is split from the ClearBoard/OpenWhiteboard code line as an independent public project and does not include textbook images, school-internal resources, or third-party branded assets.

## Features

- Blank board and local image import
- Pen, highlighter, eraser, laser pointer, and pan mode
- Undo, page management, note saving, and image/PDF export
- Unified mouse, touch, and stylus input
- Windows desktop shell with portable builds and standard Inno Setup installers

## Development

```bash
npm ci
npm run dev
npm run build
npm run release:installer
```

Installers are generated by `scripts/release.mjs` through Inno Setup 6 `ISCC.exe`. The project does not use a custom C# installer.

## Resource Boundary

This repository contains only public whiteboard code and generic assets. Do not commit textbook images, PDFs, school-internal material, or unauthorized branded assets.
