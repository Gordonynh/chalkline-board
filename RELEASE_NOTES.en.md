# Chalkline Board 0.1.0

## Added

- Split the public whiteboard into an independent project.
- Kept classroom whiteboard basics: writing, annotation, erasing, laser pointer, panning, saving, and export.
- Release script supports portable multifile builds, single-file builds, and standard Inno Setup installers.

## Changed

- Removed textbook, visualizer, and legacy custom installer entry points.
- Renamed install directory, window title, and data directory to Chalkline Board.
- Public releases exclude textbook images and internal resources.
